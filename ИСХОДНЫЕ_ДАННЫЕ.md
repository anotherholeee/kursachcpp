2.1 Структура входных и выходных данных

Входными данными программы являются файлы, содержащие необходимые данные для функционирования программы:

1. stops.txt

Хранит данные об остановках транспортной системы: уникальный идентификатор остановки и название остановки.

Формат записи в файл:

<id>|<название>

Пример записи в файл:

1|Центральный вокзал

2|Площадь Ленина

3|Улица Гагарина

2. bus.txt

Хранит данные об автобусах: тип транспорта, модель, государственный номер, вместимость пассажиров и тип топлива.

Формат записи в файл:

<тип_транспорта>|<модель>|<госномер>|<вместимость>|<тип_топлива>

Пример записи в файл:

Автобус|МАЗ-203|АН 8669-7|50|дизель

Автобус|ПАЗ-3205|ВС 1234-5|50|дизель

Автобус|ЛиАЗ-5292|АВ 2222-2|50|дизель

3. tram.txt

Хранит данные о трамваях: тип транспорта, модель, государственный номер, вместимость пассажиров и напряжение питания.

Формат записи в файл:

<тип_транспорта>|<модель>|<госномер>|<вместимость>|<напряжение>

Пример записи в файл:

Трамвай|71-931|ТР 5678-9|100|600.000000

Трамвай|71-623|ТР 3333-3|100|600.000000

Трамвай|71-407|ТР 4444-4|100|600.000000

4. trolleybus.txt

Хранит данные о троллейбусах: тип транспорта, модель, государственный номер, вместимость пассажиров и напряжение питания.

Формат записи в файл:

<тип_транспорта>|<модель>|<госномер>|<вместимость>|<напряжение>

Пример записи в файл:

Троллейбус|ЗиУ-682|ТБ 1111-1|50|600.000000

Троллейбус|Тролза-5265|ТБ 2222-2|50|600.000000

Троллейбус|ЗиУ-683|ТБ 3333-3|50|600.000000

5. drivers.txt

Хранит персональные данные водителей: имя, фамилия, отчество и категория водительских прав.

Формат записи в файл:

<имя>|<фамилия>|<отчество>|<категория>

Пример записи в файл:

Иван|Петров|Сергеевич|D

Наталья|Михайлова|Владимировна|DT

Павел|Кузнецов|Борисович|T

6. routes.txt

Хранит информацию о маршрутах: номер маршрута, тип транспорта, список остановок и дни недели работы.

Формат записи в файл:

<номер>|<тип_транспорта>|<остановка1;остановка2;...>|<день1,день2,...>

Пример записи в файл:

101|Автобус|Центральный вокзал;Площадь Ленина;Улица Гагарина;Стадион|1

202|Автобус|Центральный вокзал;Площадь Ленина;Больница;Университет|1

5|Трамвай|Парк Победы;Улица Гагарина;Больница;Университет|1

303|Троллейбус|Стадион;Улица Гагарина;Парк Победы|1

7. busTrips.txt

Хранит данные о рейсах автобусов: уникальный идентификатор рейса, номер маршрута, транспортное средство, водитель, время отправления, день недели и расписание прибытия на остановки.

Формат записи в файл:

<id>|<номер_маршрута>|<тип>|<модель>|<госномер>|<вместимость>|<тип_топлива>|<имя>|<фамилия>|<отчество>|<категория>|<время_отправления>|<день_недели>|<расписание>

Формат времени: <ЧЧ:ММ> (24-часовой формат)

Формат расписания: <остановка1>=<время1>;<остановка2>=<время2>;...

Пример записи в файл:

1|101|Автобус|МАЗ-203|АН 8669-7|50|дизель|Иван|Петров|Сергеевич|D|06:30|1|Центральный вокзал=06:30;Площадь Ленина=06:33;Улица Гагарина=06:37;Стадион=06:41

2|101|Автобус|ПАЗ-3205|ВС 1234-5|50|дизель|Мария|Сидорова|Ивановна|D|08:00|1|Центральный вокзал=08:00;Площадь Ленина=08:03;Улица Гагарина=08:07;Стадион=08:11

9|202|Автобус|ПАЗ-3205|ВС 1234-5|50|дизель|Мария|Сидорова|Ивановна|D|07:00|1|Центральный вокзал=07:00;Площадь Ленина=07:03;Больница=07:07;Университет=07:11

8. tramTrips.txt

Хранит данные о рейсах трамваев: уникальный идентификатор рейса, номер маршрута, транспортное средство, водитель, время отправления, день недели и расписание прибытия на остановки.

Формат записи в файл:

<id>|<номер_маршрута>|<тип>|<модель>|<госномер>|<вместимость>|<напряжение>|<имя>|<фамилия>|<отчество>|<категория>|<время_отправления>|<день_недели>|<расписание>

Формат времени: <ЧЧ:ММ> (24-часовой формат)

Формат расписания: <остановка1>=<время1>;<остановка2>=<время2>;...

Пример записи в файл:

16|5|Трамвай|71-931|ТР 5678-9|100|600.000000|Сергей|Новиков|Александрович|T|06:00|1|Парк Победы=06:00;Улица Гагарина=06:04;Больница=06:09;Университет=06:14

17|5|Трамвай|71-623|ТР 3333-3|100|600.000000|Анна|Морозова|Дмитриевна|T|07:30|1|Парк Победы=07:30;Улица Гагарина=07:34;Больница=07:39;Университет=07:44

36|7|Трамвай|71-623|ТР 3333-3|100|600.000000|Юлия|Орлова|Александровна|DT|08:45|1|Парк Победы=08:45;Улица Гагарина=08:49;Площадь Ленина=08:54

9. trolleybusTrips.txt

Хранит данные о рейсах троллейбусов: уникальный идентификатор рейса, номер маршрута, транспортное средство, водитель, время отправления, день недели и расписание прибытия на остановки.

Формат записи в файл:

<id>|<номер_маршрута>|<тип>|<модель>|<госномер>|<вместимость>|<напряжение>|<имя>|<фамилия>|<отчество>|<категория>|<время_отправления>|<день_недели>|<расписание>

Формат времени: <ЧЧ:ММ> (24-часовой формат)

Формат расписания: <остановка1>=<время1>;<остановка2>=<время2>;...

Пример записи в файл:

24|303|Троллейбус|ЗиУ-682|ТБ 1111-1|50|600.000000|Николай|Попов|Игоревич|T|07:15|1|Стадион=07:15;Улица Гагарина=07:18;Парк Победы=07:22

25|303|Троллейбус|Тролза-5265|ТБ 2222-2|50|600.000000|Татьяна|Васильева|Николаевна|DT|09:15|1|Стадион=09:15;Улица Гагарина=09:18;Парк Победы=09:22

56|304|Троллейбус|Тролза-5265|ТБ 2222-2|50|600.000000|Анна|Морозова|Дмитриевна|T|08:30|1|Центральный рынок=08:30;Площадь Ленина=08:33;Театр=08:37;Библиотека=08:41

10. admins.bin

Хранит учетные записи администраторов системы в бинарном формате.

Формат:

- Размер (size_t) - количество учетных записей

- Для каждой записи:

  - Размер (size_t) - длина имени пользователя

  - Строка (char[]) - имя пользователя

  - Размер (size_t) - длина пароля

  - Строка (char[]) - пароль

3. Исходные данные к проекту: записи системы управления общественным транспортом о водителях 

(имя, фамилия, отчество, категория водительских прав), транспортных средствах (тип транспорта, 

модель, номерной знак, вместимость пассажиров, тип топлива или напряжение питания), 

остановках (ID остановки, название остановки), маршрутах (номер маршрута, тип транспортного 

средства, список остановок, дни недели работы), рейсах (ID рейса, номер маршрута, ID 

транспортного средства, ID водителя, время отправления, день недели, расписание прибытия на 

остановки) и расписании работы водителей (ID водителя, список ID рейсов, назначенных водителю).

Данные хранятся в следующих файлах: водители (drivers.txt), транспортные средства (bus.txt, 

trolleybus.txt, tram.txt), остановки (stops.txt), маршруты (routes.txt), рейсы 

(busTrips.txt, trolleybusTrips.txt, tramTrips.txt).
